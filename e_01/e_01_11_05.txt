社員テーブルから部署、役職ごとに降順ソートした結果を出力する命令の訂正
作成日：6月13日
作成者：成田修之

SELECT
	depart_id,
	class,
	CONCAT('l_name','f_name') AS 氏名
FROM
	employee
ORDER BY
	depart_id,
	class DESC
;

↓↓

SELECT
	depart_id AS 部署,
	class AS 役職,
	CONCAT(l_name,f_name) AS 氏名
FROM
	employee
ORDER BY
	depart_id DESC,
	class DESC
;
